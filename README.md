# SPA Project

## Описание
Это проект для управления полезными привычками, разработанный с использованием Django и Django REST Framework.

## Установка и запуск локально

### Предварительные требования
- Python 3.13
- Poetry
- Docker и Docker Compose

### Шаги по настройке

1. **Клонируйте репозиторий**:
   ```bash
   git clone https://github.com/kseniya161-max/spa.git
2. Установите зависимости:
curl -sSL https://install.python-poetry.org | python3 -

3. Затем установите зависимости проекта
poetry install

4. Настройте переменные окружения:
Создайте файл .env в корне проекта и добавьте необходимые переменные окружения, такие как SECRET_KEY.

5. Запустите проект:
Для запуска проекта используйте Docker:
docker-compose up --build

6. Настройка CI/CD с GitHub Actions
Шаги по настройке
Создайте секреты в GitHub:
Перейдите в настройки Вашего репозитория на GitHub и добавьте следующие секреты:

DOCKER_HUB_ACCESS_TOKEN: Токен доступа к Docker Hub.
DOCKER_HUB_USERNAME: Ваше имя пользователя на Docker Hub.
SECRET_KEY: Секретный ключ для Django.
SSH_PRIVATE_KEY: Приватный SSH ключ для доступа к серверу.
SSH_USER: Имя пользователя для SSH доступа.
SERVER_IP: IP-адрес Вашего сервера.
Настройка GitHub Actions:
Убедитесь, что файл .github/workflows/ci.yml содержит правильную конфигурацию для запуска тестов, линтинга, сборки Docker-образов и деплоя на сервер.


7. Деплой на сервер
Шаги по деплою:
- Убедитесь, что сервер готов к работе с Docker и Docker Compose.
- Настройте SSH-доступ к серверу, добавив Ваш публичный ключ в файл ~/.ssh/authorized_keys на сервере.
- Запустите GitHub Actions, чтобы автоматически развернуть проект на сервере.
